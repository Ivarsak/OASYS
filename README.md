# OASYS

Requirements

- MQTT broker: "broker.hivemq.com"
- MQTT publish topic: "Oasys/Publish"
- MQTT Subscribe topic: "Oasys/Subscribe"

 For communication with the ESP8266, use a mqtt client such as mosquitto (https://mosquitto.org/)
